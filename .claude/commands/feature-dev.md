# 機能開発

新しい機能の実装を支援します。システム的なアプローチで進めます。

## フェーズ

### 1. 発見
- 構築するものを理解
- 要件を確認

### 2. コードベース探索
- 関連する既存コードとパターンを理解
- 類似機能を調査

### 3. 明確化の質問
- 曖昧点を特定
- エッジケースを確認
- ユーザーに質問を提示

### 4. アーキテクチャ設計
- 複数のアプローチを提示
- トレードオフを比較
- 推奨事項を提示

### 5. 実装
- 選択されたアプローチで実装
- コードベース規約に従う

### 6. 品質レビュー
- コードレビュー
- 改善点の特定

### 7. サマリー
- 完了した作業の記録
- 次のステップの提案

## 使用例

```
/feature-dev ユーザー認証機能を追加
```

```
/feature-dev
```
