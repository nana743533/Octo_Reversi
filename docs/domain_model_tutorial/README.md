# ドメインモデル設計チュートリアル

このチュートリアルでは、ドメイン駆動設計（DDD）の基礎概念を学び、最終的に `../domain_model.md` を理解できるようになることを目指します。

## チュートリアル構成

| ステップ | ファイル | 内容 |
|---------|---------|------|
| 1 | [01_what_is_ddd.md](./01_what_is_ddd.md) | DDDとは何か |
| 2 | [02_ubiquitous_language.md](./02_ubiquitous_language.md) | ユビキタス言語 |
| 3 | [03_bounded_context.md](./03_bounded_context.md) | バウンデッドコンテキスト |
| 4 | [04_entity_and_value_object.md](./04_entity_and_value_object.md) | エンティティと値オブジェクト |
| 5 | [05_aggregate.md](./05_aggregate.md) | 集約 |
| 6 | [06_domain_service.md](./06_domain_service.md) | ドメインサービス |
| 7 | [07_repository.md](./07_repository.md) | リポジトリ |
| 8 | [08_domain_event.md](./08_domain_event.md) | ドメインイベント |

## 学習の進め方

1. 各ステップを順番に読み進める
2. 章末の確認問題に取り組む
3. オクト対戦ゲームへの適用例を確認する
4. 全ステップ完了後、`../domain_model.md` を読む

## 前提知識

- オブジェクト指向プログラミングの基礎（クラス、メソッド等）
- 基本的なデータベース知識
- プログラミング言語（TypeScript または同等の言語）
