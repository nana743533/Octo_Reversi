# Octo_Reversi Project Context

## プロジェクト概要
Octo_Reversi は、WebベースまたはCLIベースのリバーシ（オセロ）アプリケーションです。

## 技術スタック
- 言語: 未定 (TypeScript/Python 等)
- インフラ: AWS (EC2)

## 開発ガイドライン
- コード変更は `docs/` 内の設計仕様書と整合性を保つこと。
- 新しい機能を追加する際は、まずドメインモデルとデータベースモデルへの影響を確認すること。
- テストを重視し、ロジックの変更には単体テストを伴うこと。

## Gemini 設定 (.gemini/)
このプロジェクトでは、Gemini CLI の動作をカスタマイズするために以下のディレクトリ構成を採用しています。

- `skills/`: 特定タスク（リファクタリング、テスト生成等）のための手順書。
- `commands/`: プロジェクト固有の操作スクリプトやカスタムコマンド定義。
- `rules/`: コーディング規約やアーキテクチャルールなどの詳細ルール。
- `hooks/`: 特定イベント時に実行される処理や連携スクリプト。

## ディレクトリ構造
- `docs/`: 設計ドキュメント
- `.gemini/`: Gemini CLI 設定およびコンテキスト
- `.geminiignore`: Gemini CLI 用の無視ファイル
